# 자연적 설득 구조 == 통합적 중용 대화 흐름

일상/업무/비즈니스 대화 프레임워크: "중용 기반의 사유 프레임워크 + L.E.N.S 조합"

### 핵심 요소

1. **상대 입장에 대한 '이해'가 우선** → Engage
    #LENS-Engage 
2. **이슈의 다면적 구조 파악** → Network
    #LENS-Network
3. **중심 관점 제시 (中道)** → Ship
    #LENS-Ship
4. **다시 상대와 연결해보기** → Loop
	#Loop
