## 핵심
앱이 샌드박스 격리되어 시스템의 영향이 없다.
최신 버전 앱을 제공한다 (GIMP, OBS, Inkscape 등) , 크로스 플랫폼 지원
다만 디스크 사용량이 올라가고, 초기 설치 시 세팅이 필요하다는 점?

Pop_os나 기타 리눅스 커뮤니티에서 인기가 많다.


## Pop!_OS에서 최선의 조합은?

| 용도              | 추천 패키지 매니저          |
| --------------- | ------------------- |
| 시스템, CLI 툴      | ✅ APT               |
| 최신 GUI 앱        | ✅ Flatpak (Flathub) |
| 한정된 Snap-only 앱 | Snap (보조적으로만)       |
