## 설문 조사 항목
일단 하나만 추가하고 디자인을 적용한다음 복사붙여넣기나, 단축키이용
```html
<ul class="items">
	<li class="item">
		<div class="item-id">20</div>
		<div class="item-mbti">ENTP</div>
		<div class="item-arrow">
			<img class="item-arrow-icon" src="images/arrow.svg" alt="arrow">
		</div>
		<!-- 이런 불편함이 있어 라이브러리가 필요해보임 -->
		<div class="item-color-chip" style="background: #EFB578"></div>
		<div class="item-color-code">#EFB578</div>
	</li>
</ul>
```
```CSS
.items {
```
