## 핵심
가상 클래스는 마침표대신 :으로시작하고 미리정의되어이다.

```CSS
/*
a {
	color: #12172a;
	text-decoreation: none;
}

*/

/* hover state: 올려놓고있을때*/
a:hover {
	text-decoration: underline;
}
/* active state: 꾹누르고있으면 보이는 활성상태 */
a:active {
	color: yellow
}

a:focus {
	/* 일반적으로 input에 포커스가 있는상태
		준비중인상태이므로 여기서 추가동작을 가하면 넘거가지 전 상태라고 생각하면된다.
	*/
}
/* visited state 이미 방문한 링크의 상태를 변경시킬수있다. */
a:visited {
	...
	
}
```
