## 추천
클래스 하나로 붙여서 만드는것을 권장하지만, 같은 클래스를 주다가 같은 클래스요소에 특별한 클래스를 만들고 요소를 만들어주고 싶을때 사용하는것을 추천한다.

### 추가
클래스선언시
```HTML
<div class="delay book-info">...</div>
```
HTML요소 클래스 속성에 이런식으로 사이 공백으로 목록을 만들어주는것이다
