## 핵심 특징

### 감정 기반 프로젝트 관리
- 감정은 구조적 변화의 트리거
- 에이전트의 선택 흐름을 주도
- 구조의 재배치와 최적화

### 진화하는 시스템
- 정적 루틴과 동적 개시의 조화
- 필요에 따른 행동 패턴 재구성
- 새로운 흐름의 창출

### 실험과 루프
- 실험은 파괴와 실패의 영역
- 루프는 기억과 흐름을 구축
- 프로젝트는 이 둘을 엮어냄

## 시작하기

<Steps>
    ### MetaOS 설치
    ```bash
    pnpm add metaos
    ```

    ### 기본 설정
    ```typescript
    import { MetaOS } from 'metaos'
    
    const metaos = new MetaOS({
      project: {
        name: 'My Project',
        structure: 'purpose-driven'
      },
      agent: {
        type: 'self-executing',
        capabilities: ['automation', 'learning']
      }
    })
    ```

    ### 프로젝트 시작
    ```typescript
    await metaos.initialize()
    await metaos.startProject()
    ```
</Steps>

## 핵심 구성요소

### 의도 (Intent)
- 철학과 존재 이유에 대한 질문
- 확장 가능한 구조 설계
- 지속적인 개선과 발전

### 관점 (Perspective)
- 정보와 경험을 해석하는 프레임
- 유연한 사고방식
- 다양한 시각의 통합

### 행동 (Behavior)
- 외부와의 연결성
- 실천 가능한 결과물
- 지속적인 피드백 루프
